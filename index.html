Mi Portafolio

Juego Pokemon

Aplicacion Juego de memoria Pokemon, con una API de Pokemon el juego esta creado con Html, Css y JavaScript, puedes jugar con varias personas, tambien vas subiendo de nivel (Novato, Intermedio y Avanzado) es un juego bastante practico y divertido para compartir en familia.

Aplicacion Calculadora

Aplicacion Calculadora creada con Html, Css y JavaScript tiene un diseño fresco con un uso sencillo para la comprension del usuario, con esta aplicacion se realizan operaciones matemáticas, es una máquina electrónica con la cual se pueden hacer cálculos aritméticos.

Aplicacion del Clima

Aplicacion del Clima creada con HTML, CSS y JAVASCRIPT; la aplicación de previsión metereológica pronostica el clima a diario y por hora La aplicación de pronóstico del tiempo es un canal meteorológico que tiene información meteorológica precisa

Aplicacion Super Hero

Esta es una aplicacion creada con una API de super heroes de Marvel y Dc comics donde puedes encontrar la descripcion de cada super heroe como (poderes, edad, estatura, raza etc.) para saber la descripcion del super heroe solo debes colocar el nombre y la aplicacion lo busca de inmediato, esta creada con HTML, CSS y JAVASCRIPT.

Conversor de Divisas

La Aplicacion conversor de divisas le permite comprobar los últimos tipos de cambio medios de compra y venta y convertir las principales divisas del mundo. Son tipos de cambio de divisas obtenidos de de una API, esta aplicacion esta creada con HTML, CSS y JAVASCRIPT.

Aplicacion de tareas

Añade tareas como “Leer emails del trabajo todos los días a las 10am” para hacer tu lista de pendientes en segundos utilizando el reconocimiento de lenguaje coloquial y las fechas periódicas, esta aplicacion esta creda con HTML, CSS y JAVASCRIPT
